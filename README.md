# site_web_cours
Site web avec la frameword express 
## Comment installer

Premiere chose a faire et de cloner le repertoire git :
```
git clone https://github.com/miligen2/site_web_cours.git
```

Une fois que le répertoire est cloné :
```
npm install
npm run dev
```
